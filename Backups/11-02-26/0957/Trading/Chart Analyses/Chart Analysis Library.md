## üöÄ Quick Start: How to Add a New Chart

### Method 1: Duplicate Template (Recommended)
1. Go to **Trading/Chart Analyses/** folder
2. Right-click on **Chart Analysis Template**
3. Select **"Make a copy"** (or Cmd/Ctrl+C, then Cmd/Ctrl+V)
4. Rename: `Chart Analysis - [YYYY-MM-DD] - [Brief Description]`
5. Fill in the details
6. Drag & drop chart image into the image section
7. Send to Kai via Telegram for analysis

### Method 2: Use Obsidian Templates (Advanced)
If you install the **Templater** plugin, you can:
- Press `Alt+N` (or your hotkey)
- Select "Chart Analysis Template"
- It auto-creates with today's date

## üí° Best Practices

### For Chart Images:
‚úÖ **DO:**
- Show timeframe labels clearly (4H, 1H, 5M visible)
- Include multiple timeframes if possible (split screen)
- Show key levels marked (OBs, FVGs, swing points)
- Include any annotations/lines from the original post

‚ùå **DON'T:**
- Crop out the timeframe indicator
- Show just the entry timeframe without HTF context
- Use blurry or low-resolution screenshots

### For Naming:
‚úÖ Good: `Chart Analysis - 2026-02-03 - 4H PO3 Bearish Setup`  
‚úÖ Good: `Chart Analysis - 2026-02-05 - SMT Divergence Gold`  
‚ùå Bad: `Chart1`  
‚ùå Bad: `GxT Post`

### For Organization:
- All files automatically go in `Trading/Chart Analyses/` folder
- This keeps your main Trading folder clean
- Easy to find specific examples later
